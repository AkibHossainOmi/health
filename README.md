# To Run the health project
Steps to run this project:
1. ```git clone https://github.com/asifjoardar/health.git```
2. ```cd backend```
3. ```cp .env.example .env```
   Configure your .env file as your database
4. Run `npm i` command
6. Run `npm start` command
